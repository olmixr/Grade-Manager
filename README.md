# Grade Manager (Java Console App)

This is a simple console application written in Java for managing student grades. I created this project to practice working with collections, file handling, and basic algorithms.

# Features
Add, update, remove, and view grades
Search grades by value
Sorting (Bubble Sort, Selection Sort, Insertion Sort)
Show statistics (min, max, average, count)
Generate random grades
Save and load data using serialization
Autosave and autoload settings
Undo system for main operations (add, update, remove, clear)
Technologies
Java
ArrayList
File I/O (ObjectInputStream / ObjectOutputStream)
Purpose

This project helped me understand how to work with data structures, implement sorting algorithms manually, and handle file persistence in Java.
